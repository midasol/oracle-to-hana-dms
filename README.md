# oracle-to-hana-dms

AWS DMS가 SAP HANA DB를 지원하지 않기 때문에 SAP ASE 나 MySQL, PostgreSQL 를 중계 DB로 두고 SAP HANA로 연결해야할 것 같습니다.
S3를 중계 DB로 말씀드리지 않은 경우에는 CDC 기능을 사용하는 것이 간결하지 않기 때문입니다.
